# public-image-mirror
[![FOSSA Status](https://app.fossa.com/api/projects/git%2Bgithub.com%2Fpacoxu%2Fpublic-image-mirror.svg?type=shield)](https://app.fossa.com/projects/git%2Bgithub.com%2Fpacoxu%2Fpublic-image-mirror?ref=badge_shield)


## 背景
很多镜像都在国外。比如 gcr 。国内下载很慢，需要加速。

## 目标

* 一个简洁有效的方法能够加速这些包。简洁的名称映射
* 易于添加，添加新的包，不需要去修改代码。
* 稳定可靠

## 快速开始

```
docker run -d -P docker.m.daocloud.io/nginx

```




## 使用方法

支持的镜像仓库 的 *前缀替换* 就可以使用。比如：

k8s.gcr.io/coredns/coredns => k8s-gcr.m.daocloud.io/coredns/coredns

## 替换规则

| 源站     | 替换为 |
| -------  | ------- |
| gcr.io  |  gcr.m.daocloud.io       |
| k8s.gcr.io  |  k8s-gcr.m.daocloud.io      |
| docker.io  |  docker.m.daocloud.io      |
| quay.io  |  quay.m.daocloud.io     |
| ghcr.io  | ghcr.m.daocloud.io   |

## 安装运行

安装skepo.
```
yum install skopeo --nobest
```


运行同步程序

```
export REGISTRY_PASSWORD=password #镜像仓库密码
python scripts/sync-to-jp.py
```
